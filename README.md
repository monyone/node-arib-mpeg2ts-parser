# node-arib-mpeg2ts-parser

A Simple MPEG2-TS Parser.

## Feature

* MPEG2-TS stream into TS Packet chunks

## Build

### Preparing

```bash
git clone https://github.com/monyone/node-arib-mpeg2ts-parser
cd node-arib-mpeg2ts-parser
yarn
```

### Compiling aribb24.js library

```bash
yarn run build
```
